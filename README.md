# 🛒 E-Commerce Platform

A scalable and modern **E-Commerce Web Application** built with **Next.js**, focused on performance, SEO, and real-world features like authentication, cart management, and online payments.

---

## 🚀 Live Demo

👉 https://e-commerce-amber-delta-66.vercel.app/

---

## 🎯 Overview

This project simulates a real-world e-commerce experience where users can browse products, manage their cart, authenticate securely, and complete purchases using online payment methods.

Built using **Next.js** to leverage server-side rendering, routing, and performance optimizations.

---

### 🛍️ Shopping Experience

* Browse products with modern UI

## ⚙️ Installation & Setup

1. Clone the repository:

```bash id="v7q1mz"
git clone https://github.com/KerolosAdly59/e-commerce.git
```

2. Navigate to project folder:

```bash id="a9p3ld"
cd e-commerce
```

3. Install dependencies:

```bash id="j2x9qs"
npm install
```

4. Run development server:

```bash id="m8z4kt"
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env.local` file:

```env id="m2x9pl"
NEXT_PUBLIC_API_URL=your_api_url
NEXT_PUBLIC_AUTH_KEY=your_auth_key
NEXT_PUBLIC_PAYMENT_KEY=your_payment_key
```
